# Bots

This folder groups Telegram bots and future automation bot projects.

## Conventions

- One bot per subfolder
- Each bot should keep its own `README.md`, `CLAUDE.md`, `ARCHITECTURE.md`, `PROJECT_PLAN.md`, `TASK_QUEUE.md`, and `DECISIONS.md`
- Secrets and runtime environments stay inside the bot project, not in the collection folder
