# Store Expense Bot Product Spec

## Goal

Allow a store operator to log expenses from Telegram in the fastest possible way.

## Primary User

- Store owner or operator tracking daily expenses

## Core Actions

- Quick-add expense from free text
- Guided add flow
- Daily summary
- Recent entries list

## Success Criteria

- Entry should feel faster than opening a spreadsheet manually
- Logged data should land in Google Sheets reliably
