# Websites

This folder groups website projects.

## Conventions

- One website per subfolder
- Each website should keep its own `README.md`, `CLAUDE.md`, `ARCHITECTURE.md`, `PROJECT_PLAN.md`, `TASK_QUEUE.md`, and `DECISIONS.md`
- Do not share dependencies or configs across websites unless intentionally extracted later
