# Binary Ventures Website

Marketing website for `binaryventures.in`.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- GSAP
- Lenis

## Run Locally

```bash
cd Websites/binaryventures.in
npm install
npm run dev
```

## Build

```bash
cd Websites/binaryventures.in
npm run build
```
