# Binary Ventures Website Workflows

## Local Development

1. Work inside `Websites/binaryventures.in/`
2. Run `npm run dev`
3. Review desktop and mobile behavior before merging visual changes

## Content Update Workflow

1. Update copy/data in `src/data/`
2. Adjust presentation in section components only when needed
3. Check that nav, section flow, and CTA still feel coherent

## Deployment Workflow

1. Run `npm run build`
2. Review output and hosting configuration
3. Publish using the chosen deployment target
