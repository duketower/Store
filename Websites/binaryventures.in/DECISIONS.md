# Binary Ventures Website Decisions

## Decision 001

Use a dedicated website project under `Websites/binaryventures.in/`.

Reason:
- Keeps website work isolated from POS and bot development

## Decision 002

Use a content-driven structure with data files and reusable sections.

Reason:
- Makes site copy and showcased work easier to update without rewriting page structure

## Decision 003

Keep the site visually expressive instead of default template styling.

Reason:
- The site is itself a proof of design and implementation capability
